# 🏗️ Data Engineering: Building the Data Warehouse

## 🎯 Objective

To design and build a modern data warehouse using **SQL Server**, consolidating data from multiple sources into a clean and analysis-ready format.

---

## ⚙️ Project Specifications

### 📥 Data Sources

* ERP System (CSV files)
* CRM System (CSV files)

---

### 🧹 Data Cleaning

* Removed duplicates
* Standardized inconsistent formats
* Handled missing values
* Ensured data accuracy and consistency

---

### 🔗 Data Integration

* Combined ERP and CRM datasets
* Created a unified schema for analysis
* Established relationships between tables

---

### 📊 Data Model

Designed a simple and efficient **Star Schema**:

* **Fact Table**

  * Sales (transactions, revenue, quantity)

* **Dimension Tables**

  * Customers
  * Products

This structure allows fast and flexible analytical queries.

---

### 📌 Scope

* Focused on the **latest dataset only**
* No historical tracking (no slowly changing dimensions)

---

### 📄 Documentation

* Clear table definitions
* Column descriptions
* Relationship mapping
